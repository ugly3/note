### 安装、更新与卸载软件

#### 核心命令速查表

| 命令                      | 功能描述               |
| ----------------------- | ------------------ |
| `sudo pacman -S <包名>`   | 安装指定软件（从官方仓库）      |
| `sudo pacman -Syu`      | 同步仓库并更新系统          |
| `sudo pacman -R <包名>`   | 卸载软件（保留依赖）         |
| `sudo pacman -Rs <包名>`  | 卸载软件并移除其未被其他包依赖的依赖 |
| `sudo pacman -Rns <包名>` | 彻底卸载（移除软件、依赖及配置文件） |
