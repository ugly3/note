# GD32F407VET6 开发学习笔记

## 1. 芯片概述

GD32F407 系列是由 **兆易创新 (Gigadevice)** 推出的高性能微控制器，基于 **ARM® Cortex®-M4** 内核。它在处理能力、功耗和外设资源上具有显著优势，是工业控制、医疗设备和智能硬件的理想选择。
[[SCH_核心板.pdf#page=1&annotation=annot_p0_1|SCH_核心板, 页面 1]]

## 2. 核心规格参数

| 项目        | 详细参数             | 备注            |
| --------- | ---------------- | ------------- |
| **内核**    | ARM Cortex-M4    | 带 FPU（浮点运算单元） |
| **主频**    | **168 MHz**      | 高性能处理能力       |
| **Flash** | 512 KB / 1024 KB | 取决于具体型号       |
| **SRAM**  | **192 KB**       | 充足的运行内存       |
| **工作电压**  | 2.6V ~ 3.6V      | 标准 3.3V 供电    |
| **工作温度**  | -40°C ~ +85°C    | 工业级标准         |

## 3. 存储器结构

- **Flash 存储器**：用于存放固件代码，支持指令预取缓冲器以提高执行效率。
    
- **SRAM**：
    
    - **SRAM0 (112KB)** & **SRAM1 (16KB)**：挂在总线矩阵上，供 CPU 和 DMA 使用。
        
    - **TCMSRAM (64KB)**：紧耦合内存，专门用于存放对性能要求极高的临界数据。
        

## 4. 时钟与总线架构

GD32F407 采用多层 AHB 总线矩阵，确保 CPU 和 DMA 可以并行访问不同的外设和存储器。

- **AHB 总线**：最高频率可达 168 MHz。
    
- **APB2 总线**：高速外设总线（最高 84 MHz），常用于 GPIO、高级定时器、ADC、SPI1。
    
- **APB1 总线**：低速外设总线（最高 42 MHz），常用于 通用定时器、UART、I2C、CAN、DAC。
    

## 5. 核心外设资源

- **GPIO**：支持多种模式（输入、输出、复用、模拟），具有极高的配置灵活性。
    
- **定时器**：
    
    - 高级定时器（Timer 0/7）：支持死区控制，适用于电机驱动。
        
    - 通用定时器：用于 PWM 输出、输入捕获等。
        
- **通信接口**：
    
    - **USART/UART**：支持硬件流控。
        
    - **I2C/SPI**：标准串行协议。
        
    - **CAN 2.0B**：工业常用现场总线。
        
    - **USB OTG**：支持全速/高速模式。
        
    - **Ethernet MAC**：带专用 DMA，支持 MII/RMII 接口。
        
- **模拟外设**：
    
    - **ADC**：12位高精度，支持多通道扫描。
        
    - **DAC**：12位数字转模拟。
